# Mihir Makwana — Portfolio
**Live site:** [mihir-makwana.co.uk](https://mihir-makwana.co.uk/)
**Stack:** React, TypeScript, Vite, Tailwind CSS, Framer Motion, Three.js, Supabase.
Single-page site: projects, live GitHub stats, journey, contact form (Supabase), and a small FAQ-style AI chat.
The AI chat uses Groq's `groq/compound` model. If you see a 403 permissions error, enable that model at Settings → Organization → Limits in the Groq console.
